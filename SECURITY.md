# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.** Instead, use [GitHub's private vulnerability reporting](https://github.com/mkeguy106/livestream.list.qt/security/advisories/new).

You should receive an acknowledgment within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy applies to the Livestream List Qt application and its build/release infrastructure. Third-party dependencies (PySide6, aiohttp, yt-dlp, etc.) should be reported to their respective maintainers.

## Supported Versions

Only the latest release is supported with security updates.
